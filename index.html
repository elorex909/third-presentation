Why Loops? The DRY Principle

Don't Repeat Yourself

Programming hates repetition. If you need to do something 1000 times, you shouldn't write 1000 lines.

A Loop allows us to execute a block of code repeatedly based on a specific condition.

// Bad Practice
cout << "Hello";
cout << "Hello";
cout << "Hello"; 
// ... x100 times

// Good Practice (Loop)
while (counter < 100) {
    cout << "Hello";
}

Real Life Analogy

The Track Runner

Imagine a coach tells a runner: "Run 5 laps around the track."

  • Initialization: Start at lap 0.
  • Condition: Have I run less than 5 laps?
  • Action: Run.
  • Update: Add 1 to my lap count.

1. The While Loop

The "Pre-Check" Loop

The while loop checks the condition before executing the code block.

If the condition is false initially, the code will never run.

while (condition) {
    // Code to run
    // Update step
}

While Loop Example

Print 1 to 5

We initialize a variable i outside the loop.

Inside, we print i and then increment it.

int i = 1;

while (i <= 5) {
    cout << i << " ";
    i++; // Crucial Step!
}

// Output: 1 2 3 4 5

Danger: Infinite Loops

The Trap

If you forget the update step (like i++), the condition i <= 5 will ALWAYS be true.

The program will freeze or crash (Time Limit Exceeded).

int i = 1;

while (i <= 5) {
    cout << i;
    // We forgot i++ !!
    // i stays 1 forever.
}

2. The Do-While Loop

The "Post-Check" Loop

Checks the condition after executing the code.

Guarantee: The code runs at least ONE time, even if the condition is false.

int x = 10;

do {
    cout << "Printed once!";
} while (x < 5);

// x is NOT less than 5,
// but it printed anyway.

While vs. Do-While

While

  • Checks first.
  • Might not run at all.
  • Best for: "Run as long as X is true".

Do-While

  • Runs first, checks later.
  • Runs at least once.
  • Best for: User Menus / Validation.

3. The For Loop (The King)

Compact Structure

Most popular in Competitive Programming because it organizes the 3 main loop parts in one line.

for (Init; Condition; Update) {
    // Body
}

For Loop Breakdown

  • 1. Initialization: int i = 0; (Runs once at start).
  • 2. Condition: i < n; (Checked before every iteration).
  • 3. Update: i++ (Runs after every iteration).
for (int i = 0; i < 5; i++) {
    cout << i << endl;
}

Decrementing (Countdown)

Counting Down

Loops don't just go up. They can go down.

Change the update to i-- and check if i >= 0.

// Blast off!
for (int i = 10; i >= 0; i--) {
    cout << i << " ";
}

// Output: 10 9 8 ... 0

Pattern: Accumulator

Sum of Numbers

To calculate 1 + 2 + ... + N.

Create a variable sum outside the loop starting at 0. Add i to it in every step.

int sum = 0;
int n = 5;

for (int i = 1; i <= n; i++) {
    sum += i; // sum = sum + i
}

cout << sum; // 15

Control: Break Statement

Emergency Stop

break immediately terminates the loop, skipping any remaining code in the block and any remaining iterations.

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // Stop here!
    }
    cout << i << " ";
}
// Output: 0 1 2 3 4

Control: Continue Statement

Skip This One

continue skips the rest of the code for the current iteration and jumps straight to the update step (i++).

for (int i = 0; i < 5; i++) {
    if (i == 2) {
        continue; // Skip 2
    }
    cout << i << " ";
}
// Output: 0 1 3 4

Nested Loops

Loop inside a Loop

Imagine a clock. The minute hand loops 60 times for every 1 loop of the hour hand.

This is an Outer Loop and an Inner Loop.

Grid Coordinates

Printing a Grid

i controls Rows.

j controls Columns.

for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 2; j++) {
        cout << "(" << i << "," << j << ") ";
    }
    cout << endl;
}
/* Output:
(1,1) (1,2) 
(2,1) (2,2)
(3,1) (3,2) */

Problem: Triangle of Stars

Classic Problem

Print a triangle where row 1 has 1 star, row 2 has 2 stars...

Logic: The inner loop runs from 1 to i.

int n = 4;
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= i; j++) {
        cout << "*";
    }
    cout << endl;
}
/*
*
**
***
****
*/

Concept: Variable Scope

Local vs. Global

If you declare int i inside the loop for(int i...), it only exists inside that loop.

You cannot use i after the closing curly brace }.

for (int i = 0; i < 5; i++) {
    // i works here
}

cout << i; // ERROR! 
// 'i' was destroyed.

Practice Problem 1

Easy

Even Numbers

Problem: Read an integer N. Print all even numbers between 1 and N inclusive, separated by a space.

Input: 10

Output: 2 4 6 8 10

Approach / Hint

You can do this in two ways:

  1. Loop from 1 to N and use if (i % 2 == 0).
  2. Better: Start loop at 2, and increment by 2 (i += 2).
for(int i=2; i<=n; i+=2) {
   cout << i << " ";
}

Practice Problem 2

Easy

Multiplication Table

Problem: Read an integer N. Print the multiplication table of N from 1 to 12.

Input: 5

Output:
5 * 1 = 5
...
5 * 12 = 60

Approach / Hint

Use a loop that runs 12 times (from 1 to 12).

for(int i=1; i<=12; i++) {
   cout << n << " * " << i 
        << " = " << n*i << endl;
}

Easy

Countdown

Input: 5

Output: 5 4 3 2 1

Approach / Hint

This requires a decrementing loop.

  • Start: i = n
  • Condition: i >= 1
  • Update: i--

Practice Problem 4

Medium

Factorial

Problem: Calculate N! (Factorial).
N! = 1 * 2 * ... * N.

Input: 5

Output: 120

Approach / Hint

Important: Factorials grow very fast. 20! exceeds integer limits.

Use long long for the result variable.

long long fact = 1;
for(int i=1; i<=n; i++) {
   fact = fact * i;
}

Practice Problem 5

Hard

Prime Check

Problem: Check if number N is Prime. A prime number is only divisible by 1 and itself.

Input: 7 -> Output: YES

Input: 10 -> Output: NO

Approach / Hint

Loop from 2 up to N-1.

If N is divisible by any of these numbers (remainder is 0), then it is NOT prime.

bool isPrime = true;
for(int i=2; i<n; i++) {
   if(n % i == 0) {
      isPrime = false;
      break;
   }
}

Practice Problem 6

Hard

Sum of Digits

Problem: Read an integer N. Print the sum of its digits.

Input: 123

Output: 6 (1+2+3)

Approach / Hint

Use a while loop to process the number digit by digit.

  • N % 10 gets the last digit.
  • N / 10 removes the last digit.
int sum = 0;
while(n > 0) {
   sum += n % 10;
   n /= 10;
}
